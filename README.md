# VVS Dost

A separate hackathon build derived from the Sakhi codebase, rebranded for Vasant Valley School students.

## What changed
- Separate repository and history from the original Sakhi project
- Red and beige visual theme
- VVS Dost logo assets
- Standalone in-app chat page that calls Gemini directly from the client for hackathon use

## Important note
This build intentionally exposes the Gemini API key in the frontend for speed during a hackathon demo. It is not safe for public release.
