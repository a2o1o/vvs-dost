# VVS Dost

A separate hackathon build derived from the Sakhi codebase, reworked into a student-first support guide for Vasant Valley School.

## What changed
- Separate repository and history from the original Sakhi project
- Red and beige visual theme
- VVS Dost logo assets
- Fixed decision-tree support flow instead of open-ended mentoring flows
- Rubric-based responses for stress, friendship issues, overwhelm, loneliness, embarrassment, anger, and worry about a friend
- Emergency support mode for harm, abuse, or unsafe situations
- No self-training from live student chats

## Important note
This build intentionally exposes the Gemini API key in the frontend for hackathon speed. It is not safe for public release.
